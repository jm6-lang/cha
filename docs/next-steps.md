# 后续路线图

> 完整部署方案见 [deployment.md](deployment.md)。

## Phase 0 - 合规前置（必须先做，耗时 1 个月）

> ⚠️ **必须先完成 ICP 备案才能接入微信支付/支付宝**

- [ ] 购买域名（shucha.com）
- [ ] 购买阿里云 ECS（国内节点）
- [ ] 阿里云备案系统提交 ICP 备案
- [ ] 等待管局审核（1-20 工作日）
- [ ] ICP 备案通过后，30 日内完成公安网安备案
- [ ] 网站底部添加 ICP 备案号 + 公安备案号
- [ ] 编写《用户协议》《隐私政策》
- [ ] 申请微信支付商户号 + 配置 H5 支付域名
- [ ] 申请支付宝企业账号 + 配置域名白名单
- [ ] 配置 HTTPS（SSL 证书）

## Phase 1 - 真实数据源对接（1-2 周）

- [ ] 接入公开号码归属地 API（如阿里云、聚合数据）
- [ ] 商务对接三大运营商标记数据
- [ ] 实现 `AliyunNumberLabelServiceImpl` / `TencentNumberLabelServiceImpl`
- [ ] 通过配置切换数据源 `shucha.query.provider=aliyun|tencent|mock`
- [ ] 用户登录态绑定历史（`X-Shucha-Token` 替代直接传 userId）
- [ ] 完善异常处理 + 错误码
- [ ] 单元测试 + 集成测试

## Phase 2 - 增值服务 + 商业化（2-4 周）

- [ ] `func-category` 功能商城（"查婚姻/查身份/查老赖"等作为商品）
- [ ] 会员等级体系（VIP1-VIP5，权益差异化）
- [ ] 钱包 + 充值
- [ ] 微信支付 / 支付宝集成（需 Phase 0 备案完成）
- [ ] 订单系统（订单生成、支付、查询）
- [ ] 短信验证码真实发送（阿里云 SMS）
- [ ] 限流（Sentinel）防刷

## Phase 3 - 社区 + 代理（1-2 月）

- [ ] IM 群聊（WebSocket + Netty）
- [ ] 帖子 / 动态
- [ ] 互助群 / 同城群
- [ ] 寻人 / 防骗 / 标记消除
- [ ] 代理分销（三级分销 + 佣金结算）

## Phase 4 - 高级查询服务（2-3 月）

70+ 增值查询，需要逐个对接数据源：

- [ ] 车辆查询（车牌、违章、车型）
- [ ] 企业查询（工商、股东、风险）
- [ ] 金融查询（信用卡、失信）
- [ ] 身份查询（身份证 OCR、风险）
- [ ] 婚姻记录
- [ ] 商标 / 专利
- [ ] 司法诉讼 / 老赖

每个查询都是：1) 数据源商务 → 2) 接口对接 → 3) DTO/Service → 4) 前端页面

## Phase 5 - 生产化部署（1 月）

详见 [deployment.md](deployment.md) 完整方案。
